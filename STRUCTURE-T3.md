# TCF Canada — Tâche 3 : Structure & Phrases

## Format: 150-180 words | Time: 22-25 minutes

---

## The structure is ALWAYS the same (3 paragraphs)

```
P1: Synthèse des documents
P2: Opinion personnelle + arguments
P3: Nuance + conclusion
```

---

## P1 — Synthèse (write on AUTOPILOT)

### Opening sentence (always the same)

```
Le sujet sur [X] oppose deux visions divergentes.
```

or

```
Le sujet de [X] oppose deux visions divergentes.
```

### Document 1

```
D'un côté, le premier document [VERB] [résumé].
```

**Verbs to choose from:**

| Verb | Usage |
|---|---|
| *souligne* | highlights (neutral) |
| *présente* | presents (neutral) |
| *met en garde contre* | warns against (negative topic) |
| *soutient que* | argues that (opinion) |
| *affirme que* | states that (strong) |

### Document 2

```
De l'autre, le deuxième [VERB] [résumé].
```

**Alternative with "tandis que" (more fluid):**

```
D'un côté, le premier document souligne [X],
tandis que le deuxième soutient que [Y].
```

### Useful phrases for summarizing

- *les bénéfices apportés par...*
- *les bienfaits psychologiques / sanitaires tels que...*
- *les impacts environnementaux négatifs comme...*
- *nous rappelle les investissements en temps et en argent*
- *en présentant la limite de...*

## P2 — Opinion personnelle (your SIGNATURE structure)

### Opening — the concessive subjunctive (use EVERY time)

```
Personnellement, si [ADJ] que soient/soit les/le [NOM],
on ne saurait [VERBE INFINITIF] [contre-argument].
```

**Adjective bank:**

| Adjective | For topics about |
|---|---|
| *attrayants* | benefits, advantages |
| *importante* | a cause, an issue |
| *néfastes* | negative impacts |
| *pratique* | a convenient solution |
| *séduisants* | an appealing idea |
| *préoccupants* | worrying trends |

**Verb bank (after "on ne saurait"):**

| Verb | Meaning |
|---|---|
| *ignorer que...* | ignore that... |
| *sous-estimer...* | underestimate... |
| *arrêter...* | stop... |
| *négliger...* | neglect... |
| *nier que...* | deny that... |

### Supporting arguments

**First argument:**
```
En effet, [explication + exemple].
```

**Second argument (pick one connector):**

| Connector | Tone |
|---|---|
| *De surcroît,* | Moreover (formal, impressive) |
| *Par ailleurs,* | Furthermore |
| *Par conséquent,* | Consequently (cause-effect) |
| *De plus,* | In addition (simpler) |

### Useful argumentative phrases

- *la majorité des [gens] n'a pas de compétences suffisantes pour...*
- *ces exigences constantes amènent souvent [qqn] à reconsidérer leur position*
- *la presse regorge de nouvelles sur...*
- *l'expansion de [X] entraîne de nombreux bénéfices, notamment [A], [B] et [C], entre autres*
- *ces biais sont souvent négligés lors des sondages*
- *des statistiques trompeuses cachant les problématiques réelles*

### Government / society action phrases

- *Le gouvernement doit mettre en place des mesures pour...*
- *Il est essentiel que les autorités agissent pour...*
- *Il importe de s'affranchir de toutes les discriminations*
- *Les pouvoirs publics doivent intervenir pour...*

## P3 — Nuance + Conclusion (AUTOPILOT)

### Concession (acknowledge the OTHER side)

```
Certes, [opposing point] est/sont/reste(nt) indéniable(s).
```

**Examples:**
- *Certes, les risques de crimes numériques sont indéniables.*
- *Certes, l'étalement urbain incontrôlé reste une menace.*
- *Certes, la nécessité de lutter contre les discriminations est indéniable.*

### Counter (come back to YOUR point)

```
Toutefois, encore faut-il reconnaître [nuance / other perspective].
```

**or with reformulation:**

```
Toutefois, encore faut-il reconnaître [X].
Autrement dit, [reformulate in different words].
```

### Conclusion (pick one pattern)

**Pattern 1 — prescriptive (most common):**
```
En somme, [actors] doivent [action] afin de [goal].
```

**Pattern 2 — "the key is":**
```
En somme, [solution] est la clé pour [goal].
```

**Pattern 3 — reframe the debate:**
```
En somme, ce n'est pas [X] qu'il faut combattre,
mais [Y].
```

**Useful conclusion phrases:**
- *les pouvoirs publics et les citoyens doivent unir leurs efforts afin de...*
- *l'éducation numérique accompagnée d'un certain niveau de prudence est la clé pour...*
- *une analyse approfondie doit être menée avant de...*
- *l'égalité de choix doit être favorisée pour permettre à...*
- *il est essentiel de sensibiliser les citoyens aux...*

## Complete autopilot template

```
Le sujet sur [X] oppose deux visions divergentes. D'un
côté, le premier document [verbe] [résumé doc 1]. De
l'autre, le deuxième [verbe] [résumé doc 2].

Personnellement, si [adj] que soient les [nom], on ne
saurait [verbe] [contre-argument]. En effet, [argument 1].
De surcroît / Par ailleurs, [argument 2].

Certes, [concession OTHER side] est/sont indéniable(s).
Toutefois, encore faut-il reconnaître [nuance]. En somme,
[conclusion prescriptive].
```

## Connectors cheat sheet (vary these — don't always use the same)

| Function | Options |
|---|---|
| Adding | *De plus, Par ailleurs, De surcroît, En outre* |
| Cause | *En effet, Par conséquent, Cela entraîne* |
| Concession | *Certes... Toutefois, Bien que... il n'en reste pas moins* |
| Contrast | *Tandis que, Alors que, En revanche* |
| Reformulation | *Autrement dit, En d'autres termes* |
| Examples | *Notamment, Tels que, En particulier, Entre autres* |
| Conclusion | *En somme, En fin de compte, En définitive* |

## The concession flip (important!)

**Wrong pattern (conceding your OWN point):**
```
YOU: urbanisation can be positive
CERTES: Certes, les avantages sont indéniables ← your point!
```

**Correct pattern (conceding the OTHER side):**
```
YOU: urbanisation can be positive
CERTES: Certes, l'étalement urbain reste une menace ← other side
TOUTEFOIS: but planned urbanisation is the solution ← back to you
```
