# French Prepositions — Common Errors

Prepositions that I frequently get wrong. Check this before writing.

---

## Verbs + Preposition

| Verb | Correct | Wrong | Example |
|---|---|---|---|
| s'intéresser | **à** | ~~sur~~ | *Merci de vous intéresser **à** mon appartement.* |
| se limiter | **à** | ~~de~~ | *Son talent ne se limite pas **à** sa voix.* |
| se distinguer | **par** | | *Il se distingue **par** sa capacité à...* |
| se protéger | **de / des** | | *Se protéger **des** actes malveillants.* |
| participer | **à** | | *Participer **à** un jeu télévisé.* |
| se fier | **à** | | *Se fier **aux** rendez-vous médicaux.* |
| regorger | **de** | ~~des~~ | *La presse regorge **de** nouvelles.* |
| apprendre (about) | **sur** or **en** apprendre + **sur** | ~~direct object~~ | *En apprendre davantage **sur** l'université.* |
| raconter | direct object (no prep) | ~~dire de~~ | *Raconter **ma** première semaine.* |
| parler | **de** | | *Parler **de** ma première semaine.* |
| introduire | **dans** | ~~à~~ | *Introduire un membre **dans** la famille.* |
| se concentrer | **sur** | ~~aux / à~~ | *Me concentrer **sur** les tâches.* |
| continuer | **à / de** + infinitive | ~~direct infinitive~~ | *Continuer **à ne pas** utiliser Instagram.* |
| recommander | **de** + infinitive | ~~direct infinitive~~ | *Je vous recommanderais **de** tenter le coup.* |

## Expressions + Preposition

| Expression | Correct | Wrong | Example |
|---|---|---|---|
| merci + verb | **de** + infinitive | ~~pour~~ | *Merci **de** vous intéresser...* |
| merci + noun | **de** or **pour** (both OK) | | *Merci **de** ton aide.* / *Merci **pour** ton aide.* |
| remercier + verb | **de** + infinitive | | *Je te remercie **d'**être venu.* |
| remercier + noun | **de** or **pour** | | *Je te remercie **de** ton aide.* |
| à deux pas | **de** | | *À deux pas **de** la station.* |
| à proximité | **de** | | *À proximité **de** la gare.* |
| au sujet | **de** | | *Au sujet **des** détails.* |
| face | **à** | | *Face **à** des milliers de fans.* |
| grâce | **à** | | *Grâce **à** Facebook.* |

## Quick Rules

### Merci: DE for verbs, DE or POUR for nouns
- *Merci **de** m'avoir aidé.* (verb → de)
- *Merci **de** ton aide.* / *Merci **pour** ton aide.* (noun → both OK)
- NEVER: ~~Merci pour m'avoir aidé~~

### S'intéresser: always À, never SUR
- *Je m'intéresse **à** la musique.* (not ~~sur la musique~~)

### Capacité: always À, never DE
- *La capacité **à** comprendre.* (not ~~de comprendre~~)

### Rempli: always DE, never PAR
- *Rempli **de** joie.* (not ~~par la joie~~)

### Côté: NO article after informal "côté"
- *Côté **culture**, ...* (not ~~Côté la culture~~)

### DE for characteristics (size, colour, shape), not EN
- *de tailles variées* (not ~~en tailles variées~~)
- *de couleurs différentes*

### Parler DE: repeat DE for each noun
- *parler **de** mon pays **et de** sa culture* (not ~~et sa culture~~)

### ENTRE (small group 2-3) vs PARMI (large group)
- *une ambiance conviviale **entre** nous* (3 flatmates — not ~~parmi nous~~)
- ***Parmi** les centaines de participants...*

### À cause de vs EN RAISON DE vs GRÂCE À
- **À cause de** = negative/blame: *à cause de la pluie*
- **En raison de** = neutral: *en raison des loyers élevés* (not ~~à cause des~~)
- **Grâce à** = positive: *grâce à mes amis*
