# TCF Canada — Tâche 2 : Structure & Phrases

## Format: 120-150 words | Time: 18-20 minutes

---

## Step 1: Identify the frame (10 seconds)

| Consigne type | Frame | Example topics |
|---|---|---|
| Raconter une expérience | **A** | Voyage, fête, première semaine, compétition |
| Problème / mésaventure | **B** | Réclamation, incident, mauvaise expérience |
| Opinion / appréciation | **C** | Artiste préféré, ville, invention, activité |

---

## Step 2: 2-minute plan on brouillon

```
QUOI:  _____ (one word)
OÙ:    _____ (one word)
QUAND: _____ (one word)
AVEC:  _____ (one word)
FRAME: A / B / C

P1: [opening + emotional label] → 2 sentences
P2: [story beats + highlight] → 5-6 sentences
P3: [reflection + closing] → 2 sentences
```

## FRAME A: Raconter une expérience

### Opening (pick one — write without thinking)

| Phrase | Usage |
|---|---|
| *Je tenais à vous raconter [quoi].* | Warm, personal — best default |
| *Je tenais à vous parler de [événement].* | Slightly softer |
| *Je voulais partager avec vous [quoi].* | Simple alternative |

### Emotional label (follows the opening)

| Phrase | Tone |
|---|---|
| *Ce fut une expérience enrichissante et inoubliable.* | Positive, general |
| *C'était une soirée riche en rires et en souvenirs inoubliables.* | Festive |
| *Ce fut une expérience à la fois stimulante et enrichissante.* | Nuanced |
| *C'était une expérience très libératrice.* | Transformative |

### Story beats — time markers

```
Dans un premier temps, ... [action 1 — passé composé]
[Description — imparfait: ambiance, feelings, setting]
Par la suite, ... [action 2 — passé composé]
[Detail or consequence]
```

**Alternative time markers:**
- *Dès le premier jour / Dès mon arrivée, ...*
- *Au début, j'étais un peu nerveux/stressé, mais ensuite...*
- *Après [X], ...*
- *En fin de soirée / Pour finir, ...*

### Useful body phrases

| Phrase | Usage |
|---|---|
| *L'ambiance était conviviale et bienveillante, ce qui a grandement facilité...* | Describing atmosphere |
| *Composé(e) d'une dizaine de [personnes].* | Quantifying a group |
| *Étant donné que [raison], il/elle a...* | Explaining why (B2 connector) |
| *On m'a confié plusieurs tâches, notamment...* | Work/school context |
| *Un menu sur mesure correspondant aux préférences de chacun.* | Participe présent (B2) |
| *Les sessions étaient riches en informations utiles.* | School/work context |

### Highlight (THE key B2 sentence)

| Phrase | Usage |
|---|---|
| *Ce qui m'a le plus marqué, c'est [moment/détail].* | **Default — use this one** |
| *L'un des moments forts a été quand...* | Alternative |
| *Cet instant était sans doute le moment fort de la soirée.* | For events |

### Reflection (the B2 differentiator — adds 1 sentence, huge impact)

| Phrase | Usage |
|---|---|
| *Cette expérience m'a rappelé à quel point [X] est important(e).* | Positive realization |
| *Cela m'a fait prendre conscience de l'importance de [X].* | Learning something |
| *Ce moment m'a conforté dans l'idée que [belief].* | Confirming a belief |
| *J'ai réalisé à quel point [X] comptait pour moi.* | Personal discovery |

### Closing (pick one)

| Phrase | Tone |
|---|---|
| *J'ose espérer que l'avenir nous réserve encore de tels instants.* | Elegant, hopeful |
| *J'ose espérer que la suite sera à la hauteur de ces débuts prometteurs.* | New beginnings |
| *Ces souvenirs resteront à jamais gravés en moi.* | Emotional |
| *Je vous recommanderais vivement d'en faire autant !* | With recommendation |

### Sign-off
- *Amicalement,* or *À très vite,*

## FRAME B: Problème / mésaventure

### Opening
```
Il m'est arrivé récemment une expérience que je souhaite
partager avec vous.
```

### Story structure
```
[Contexte — imparfait]
Le [moment], je [description de la situation initiale].
Tout semblait [normal / parfait / calme].

[Problème — passé composé]
Soudain, [événement inattendu].
J'ai alors [réaction].
Heureusement / Malheureusement, [conséquence].

[Résolution]
Finalement, [comment cela s'est résolu].
Grâce à [aide / action], j'ai pu [résultat].
```

### Closing
```
Cette expérience m'a appris que [leçon].
Si je devais donner un conseil, ce serait de [recommandation].
```

## FRAME C: Opinion / appréciation

### Opening
```
Dans [domaine], on trouve l'embarras du choix : [liste].
Parmi toutes ces options, [sujet] occupe une place
particulière dans mon cœur / dans ma vie.
```

### Body structure
```
[Présentation — qui/quoi]
[Sujet] est [description factuelle].
Ce qui le/la distingue, c'est [qualité principale].

[Argument 1]
Tout d'abord, [première raison + exemple].

[Argument 2]
Par ailleurs, [deuxième raison + impact].
Ce qui en fait [un choix populaire / une référence].

[Touche personnelle]
Personnellement, [comment ce sujet vous touche].
Ses mélodies / Son œuvre resteront à jamais liées à [souvenir].
```

### Closing
```
Si vous ne connaissez pas encore [sujet],
je vous le/la recommande vivement !
```
